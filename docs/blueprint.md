# **App Name**: VibeShare

## Core Features:

- Secure User Authentication: User registration, login, and session management to secure personal accounts.
- Personalized Media Feed: Display user-uploaded photos and videos in a scrollable, dynamic feed.
- Media Upload & Sharing: Enable users to upload images and videos directly from their devices for public or private sharing.
- In-App Chat Messaging: Facilitate real-time text-based chat conversations between users.
- Audio Recording & Sharing: Allow users to record and share short audio clips with friends or their feed.
- AI-Powered Caption & Hashtag Suggestions: A generative AI tool that suggests relevant captions and hashtags for uploaded media based on its content.

## Style Guidelines:

- Color scheme: Light. Inspired by concepts of 'Creativity' and 'Expression'.
- Primary color: A vibrant lavender blue (#9372DB). Chosen to convey a sense of modern creativity and warmth.
- Background color: A very light, subtle purple-grey (#F3F0F6). This desaturated shade ensures visual calm and highlights the primary color.
- Accent color: A rich, energetic blue (#285DCE). It is analogous to the primary color, offering visual interest and acting as a highlight for interactive elements.
- Font choice: 'Inter' (sans-serif) for both headlines and body text. Its modern, clean, and highly readable characteristics make it suitable for a social media application.
- Use a set of minimalist and modern line icons that are easily recognizable and intuitively convey social interaction, media sharing, and recording functionalities.
- Employ a clean, mobile-responsive card-based layout for the media feed, prioritizing content visibility and user interaction. Ensure intuitive navigation across chat, profile, and feed sections.
- Implement subtle micro-interactions on button presses and smooth transitions between different sections or content items, enhancing the user experience without being distracting.
